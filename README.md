# Development
## Requirements
This package has been developed and tested using node 8 LTS (Carbon).

## Installation
Install package requirements as follows:
`npm install`

## Commands
- `npm test`
- `npm run lint`
- `npm run watch`
- `npm run test-browser`
- `npm run build`
- `npm run coverage`
