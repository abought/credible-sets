(inner) markCredibleSetScaled(statistics, cutoffopt) → {Array.<Number>}

Analyze a set of probabilities and return a fraction saying how much each item contributes to the credible set. For example, if a single item accounts for 96% of total probabilities, then for the 95% credible set, that item would be scaled to "1.0" (because it alone represents the entire credible set and then some). This is a helper method for, eg, visualizing the relative significance of contributions to the credible set. For example, when a gradient must be specified in advance and is not auto-determined by the range of the data.
