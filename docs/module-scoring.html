(inner) bayesFactors(nlogpvals, capopt) → {Array.<Number>}

Calculate a probability statistic exp(Z^2 / 2) based on pvalues. If the Z-score is very large, the bayes factors can be calculated in an inexact (capped) manner that makes the calculation tractable but preserves comparisons
Parameters:
Name Type Attributes Default Description
nlogpvals Array.<Number> An array of -log(pvalue) entries
cap Boolean <optional>
 true Whether to apply an inexact method. If false, some values in the return array may be represented as "Infinity", but the bayes factors will be directly calculated wherever possible.

(inner) posteriorProbabilities(scores) → {Array.<Number>}

Convert an array of raw per-element probability scores into posterior probabilities, dividing by the sum of all elements in the range. This is a helper function for visualization and analysis; most of the methods in this library will convert raw scores to posterior probabilities internally.
